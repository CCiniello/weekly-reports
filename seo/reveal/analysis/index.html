7 Source-of-truth cheat sheetstatic · learning reference

A standing reference for the team: which tool is the truth for which number, and why. The agent doesn't touch this — it's here so anyone reading the brief knows where a figure came from and doesn't compare two tools that measure different things.

Keyword rankings & clicks
Google Search Console
GSC is Google's own record of how we actually rank and get clicked — ground truth for pages that already rank. We do NOT use Ahrefs for our own positions ✕ Ahrefs because Ahrefs positions are modeled/sampled estimates on a schedule, while GSC is the real, daily, per-query data from Google.
Keywords we don't rank for yet
Ahrefs Keywords Explorer + Rank Tracker
The one place Ahrefs is the source of truth: search volume, difficulty, and finding terms we don't yet appear for. GSC can't show these ✕ GSC — it only reports queries we already surface on, so it's blind to whitespace. Volume/difficulty always come from Ahrefs.
Sessions, engagement, on-site conversions
GA4 (via Supermetrics)
GA4 measures what visitors do once on the site; GSC measures what happens in Google's results. They'll never match exactly — use GSC for search performance, GA4 for on-site behavior, and don't reconcile the two.
Leads / MQLs (organic-sourced)
Salesforce
Salesforce is the only system that ties a lead to revenue reality. Lead counts come from the Lead Event object filtered to organic + Reveal — never from GA4 “conversions,” which are on-site proxies, not qualified leads.
AI / LLM visibility
Amplitude AI Visibility — single source
Amplitude is the single source of truth for anything LLM/AEO — mentions %, citations, competitor share of voice, sentiment. We deliberately do not mix in Ahrefs Brand Radar ✕ Brand Radar for the weekly number, so the trend stays consistent (mixing tools = fake movement). Brand Radar stays an optional manual pull for domain-level PR targets only.
Domain Rating, backlinks, referring domains
Ahrefs Site Explorer
Authority metrics (DR, refdomains, backlink growth) are Ahrefs' model and only make sense measured in Ahrefs. Don't compare DR to any other tool's authority score — different scales.
Technical health & site errors
Ahrefs Site Audit
Crawl-based health score, broken URLs, canonicals, sitemaps. Feeds the separate Technical SEO audit, not this brief — but the health number is mirrored on the scoreboard so leadership sees the foundation risk in one place.
The one rule: one metric, one tool — never blend sources for the same number. A figure's trend is only trustworthy if it comes from the same place every week. When two tools disagree, they're usually measuring two different things (search vs. on-site, real vs. modeled), not contradicting each other.
