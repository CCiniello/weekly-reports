The break point is click→session, and it invalidates everything downstream. Google Ads reports 1,741 React clicks; GA4 attributes only ~54 sessions to the same campaigns, and Search-Charts and Search-Components register zero sessions despite 369 clicks between them. Engagement is strong on the pages GA4 does see (PMax 74–75%), so this is a measurement/attribution failure — GCLID auto-tagging, the GA4 tag on the React landing pages, or the conversion import — not a landing-page quality problem. Until it's fixed, CPA and conversion goals are unreportable. This is the #1 fix and it sits ahead of any bid or budget move.
